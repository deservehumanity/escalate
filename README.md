# Escalate Wallet
Escalate is a minimalistic CLI based Bitoin wallet

## Reason to exist
This tool exists mainly as my university project, but I plan on expanding it for convinient usage not only by myself, but any other individual.

## Capabilities
1. Create wallet
2. Create addresses
3. List addresses
4. Check balance on latest address

## Notes
Escalate is under development, right now it is available as a raw and even more minimal version than intended
